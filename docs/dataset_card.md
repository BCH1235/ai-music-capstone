# 데이터셋 카드 (1.7단계)

사용 목적
- 30초 길이 BGM 제작을 위한 자체 기호 생성기(멜로디 트랜스포머 + 드럼-그루브 VAE) 학습.
- 학습 데이터는 **MIDI 전용**입니다. 결과물은 고정된 사운드폰트를 사용하여 오디오로 렌더링됩니다.

포함 기준
- 명확한 타이밍을 가진 MIDI 파일; 4/4 박자 선호; 길이 ≥ 4마디.
제외 규칙
- 명백한 라이선스 충돌; 손상된 타이밍 또는 극심한 템포 변화가 있는 악곡.

전처리 계획 (요약)
- 제한된 BPM 세트로 정규화; 16분음표로 양자화; 4/8/16마디 크롭 추출.
- 키/스케일 감지 추가; 멜로디 및 드럼 서브셋을 위한 토큰 파일 작성.

윤리 및 지적 재산권
- 연구/교육용으로만 사용; 출처 및 저작권 표기 목록을 유지합니다.
- 원본 저작권 자료는 재배포하지 않습니다.
